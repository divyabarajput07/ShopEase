# 🛍 ShopEase

ShopEase is a simple E-commerce web application built using Django. Users can browse products, add them to the cart, manage quantity, create a wishlist, and place orders.

## 🚀 Features

- User Signup & Login
- User Logout
- Product Listing
- Product Detail Page
- Search Products
- Add to Cart
- Increase / Decrease Quantity
- Remove from Cart
- Wishlist
- Checkout
- Order History
- Responsive UI using Bootstrap

## 🛠 Technologies Used

- Python
- Django
- HTML
- CSS
- Bootstrap
- SQLite
